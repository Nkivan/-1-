## Можливості програми

- встановлення бюджету
- додавання витрат
- перегляд усіх витрат
- фільтрація витрат:
  - за датою
  - за періодом
  - за категорією
- підрахунок залишку бюджету
- звіт по категоріях

## Збереження даних

Усі дані зберігаються у файлі JSON (`finance_data.json`).  
При запуску програми дані зчитуються з файлу, а після кожної зміни автоматично оновлюються.

## Алгоритм роботи

1. При запуску програми завантажуються дані з файлу.
2. Користувач вводить команду.
3. Програма виконує відповідну функцію.
4. Після змін дані зберігаються у файл.
5. Програма працює у циклі до команди **"вийти"**.
